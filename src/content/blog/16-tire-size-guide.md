## タイヤサイズ表記の例

タイヤの側面（サイドウォール）には次のような表記があります。

```
205 / 55 R 16 91 V
 ①   ②  ③  ④  ⑤ ⑥
```

それぞれの数字・記号の意味を解説します。

## ① タイヤ幅（mm）

**205** → タイヤの断面幅（最も広い部分）が **205mm** であることを示します。

一般的な乗用車では155mm〜315mmの範囲が多く、数字が大きいほど幅広なタイヤです。

## ② 扁平率（アスペクト比）

**55** → タイヤの断面高さがタイヤ幅の **55%** であることを示します。

```
断面高さ = タイヤ幅 × 扁平率 / 100
       = 205 × 55 / 100 = 112.75mm
```

- 扁平率が**低い（40・35など）** → タイヤが薄い・スポーティーな見た目・乗り心地は硬め
- 扁平率が**高い（65・70など）** → タイヤが厚い・乗り心地が柔らかい・タイヤコストが安め

## ③ 構造記号

**R** → ラジアル構造（Radial）を示します。現代の乗用車タイヤはほぼすべてラジアル構造です。

## ④ リム径（インチ）

**16** → タイヤを装着するホイールの内径が **16インチ** であることを示します。

インチアップ（例：16→17インチ）する場合は外径を維持するために扁平率を下げる必要があります。

## ⑤ ロードインデックス（荷重指数）

**91** → タイヤ1本が支えられる最大荷重を示す指数です。

| ロードインデックス | 最大負荷能力 |
|--------------|----------|
| 82 | 475kg |
| 87 | 545kg |
| 91 | 615kg |
| 95 | 690kg |
| 100 | 800kg |

車の純正タイヤと同等以上のロードインデックスが必要です。

## ⑥ 速度記号

**V** → そのタイヤが対応できる最高速度を示します。

| 速度記号 | 最高速度 |
|--------|--------|
| S | 180km/h |
| T | 190km/h |
| H | 210km/h |
| V | 240km/h |
| W | 270km/h |
| Y | 300km/h |

## タイヤの外径計算方法

```
外径（mm）= ホイール径（インチ × 25.4） + タイヤ幅 × 扁平率/100 × 2
```

**例：205/55R16**
```
外径 = 16 × 25.4 + 205 × 0.55 × 2
     = 406.4 + 225.5
     = 631.9mm（約632mm）
```

## その他の表記

### 製造年週

側面に「**2324**」のような4桁の数字 → 2024年の23週目製造を意味します。

タイヤの製造から**5〜6年**経過したものは外観に問題がなくても交換を推奨されることが多いです。

### M+S（マッドアンドスノー）

泥・雪道での使用に対応していることを示します。
スタッドレスタイヤでなくてもM+S表記がある場合があります。

### スリップサイン

タイヤ側面に「▲」マークがある箇所を確認。トレッドに刻まれた溝の深さが **1.6mm以下**になるとスリップサインが表れ、法律上走行不可となります。

| リム径（インチ） | ホイールのサイズ | 16 |
| ロードインデックス | 最大荷重能力の指数 | 91 |
